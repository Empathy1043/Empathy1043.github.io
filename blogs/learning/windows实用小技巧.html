# Windows 常用小技巧

# 快捷键大全

# 基础快捷键

快捷键 功能描述
Win + E 打开文件资源管理器
Win + D 显示/隐藏桌面
Win + L 锁定计算机
Win + R 打开“运行”对话框
Ctrl + Shift + Esc 直接打开任务管理器
Alt + Tab 切换应用程序窗口
Win + V 打开剪贴板历史记录（Win10+）

# 高级操作

  • 分屏操作：
    Win + ←/→ 将窗口吸附到屏幕左/右侧（可组合 Win + ↑/↓ 调整分屏比例）
  • 虚拟桌面：
    Win + Ctrl + D 新建虚拟桌面，Win + Ctrl + ←/→ 切换虚拟桌面
  • 快速截图：
    Win + Shift + S 打开截图工具（区域截图+编辑）

# 系统优化技巧

# 1. 加速开机

  1. 按 Win + R 输入 msconfig → 切换到【服务】选项卡 → 勾选【隐藏所有Microsoft服务】→ 禁用第三方服务
  2. 任务管理器 → 【启动】选项卡 → 禁用不必要的启动项

# 2. 清理磁盘

  • 按 Win + R 输入 cleanmgr → 选择要清理的磁盘
  • 手动清理：删除 C:\Windows\Temp 和 %temp% 文件夹中的临时文件

# 3. 找回存储空间

  • 禁用休眠文件（需管理员权限）：
    命令提示符输入 powercfg -h off
  • 压缩系统文件：
    右键C盘 → 【属性】→ 【磁盘清理】→ 【清理系统文件】→ 勾选【以前的Windows安装】

# 文件管理技巧

  • 将文件夹拖拽到文件资源管理器的左侧【快速访问】栏
  • 按 Win + R 输入 shell:startup 可管理开机启动文件夹

# 2. 批量重命名

  1. 选中多个文件 → 按 F2 → 输入名称（如“照片(1).jpg”会自动编号）
  2. 使用PowerShell脚本：
    Get-ChildItem *.jpg | ForEach-Object { Rename-Item $_ -NewName ("Vacation_{0}.jpg" -f $_.BaseName) }
    1

# 3. 显示文件扩展名

文件资源管理器 → 【查看】→ 勾选【文件扩展名】

# 隐藏功能

# 1. 上帝模式

  1. 新建文件夹 → 重命名为：
    GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}
  2. 双击打开可访问200+系统设置入口

# 2. 快速打开系统功能

  • 按 Win + R 输入以下命令：
    ms-settings: 系统设置 | control 控制面板 | devmgmt.msc 设备管理器

# 3. 强制彻底删除文件

按 Shift + Delete 直接永久删除（不经过回收站）

# 故障排查

# 1. 网络重置

  1. 按 Win + I 打开设置 → 【网络和Internet】→ 【高级网络设置】→ 【网络重置】
  2. 或在命令提示符输入：
    netsh winsock reset 和 netsh int ip reset

# 2. 修复系统文件

命令提示符（管理员）输入：
sfc /scannow
若无效则继续执行：
DISM /Online /Cleanup-Image /RestoreHealth

💡 提示：部分操作需要管理员权限，谨慎修改系统关键设置！
🔧 适用系统：Windows 10/11（部分功能可能需最新版本支持）
