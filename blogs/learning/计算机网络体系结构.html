# 计算机网络体系结构

计算机网络作为计算机基础的核心内容，其重要性不言而喻，现在梳理一下计算机网络的体系结构

# 什么是计算机网络？

计算机网络是将多个分散且自治的计算机系统相互连接，实现信息传递与资源共享的系统。

# 互连网与互联网

internet（互连网）

  • 多个计算机网络互连形成的计算机网络。
  • 可用任意通信协议

Internet （互联网）

  • 全球最大的、开放的、由众多路由器互连形成的特定计算机网络（因特网）。
  • 使用 TCP/IP协议

  • 共享
