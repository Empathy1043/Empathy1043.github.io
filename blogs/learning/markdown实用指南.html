# Markdown 实用指南

# 什么是 Markdown？

Markdown 是一种轻量级标记语言，用简洁的语法实现内容排版，可轻松转换为 HTML。适用于文档编写、笔记记录、博客撰写等场景。

# 2. 段落与换行

  • 段落：直接输入文字，段落间空一行
  • 换行：行尾添加两个空格或换行符 \
