    在 Markdown 中，在有序列表中插入代码块时，需要注意缩进规则（这个缩进格式在obsidian上面面太恶心了，一直调不好）。以下是两种常用方法：

    # 方法 1：使用 三个反引号（推荐）

    在列表项下方空一行，缩进代码块（通常为 4 个空格或 1 个制表符），然后包裹代码：

    1. 有序列表项：

    ```python
    def hello():
        print("Hello World")
    ```

2. 下一个列表项...
    1
    2
    3
    4
    5
    6
    7
    8

    # 方法 2：使用 缩进代码块

    直接缩进代码块（通常为 8 个空格，因列表项已占用 4 个空格）：

    1. 有序列表项：

        def hello():
            print("Hello World")

2. 下一个列表项...
    1
    2
    3
    4
    5
    6

    # 注意事项：

    1. 缩进一致性：确保代码块的缩进与列表项的层级对齐。
    2. 空行分隔：列表项文本和代码块之间建议空一行以提高可读性。
    3. 解析器差异：不同平台（如 GitHub、Typora）可能对缩进要求略有不同，需实测调整。

      def hello():
    print("Hello World")
      1
      2

    2. 下一个列表项...
